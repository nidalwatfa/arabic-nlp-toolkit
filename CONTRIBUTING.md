# دليل المساهمين (Contributing Guide)

شكراً لاهتمامك بالمساهمة في مشروع **Arabic NLP Toolkit** 🎉  
هذا المشروع مفتوح المصدر ونرحب بجميع أنواع المساهمات سواء كانت إصلاح أخطاء، إضافة ميزات جديدة، أو تحسين التوثيق.

---

1. **اعمل Fork للمستودع**  
   - اضغط على زر "Fork" في صفحة المشروع على GitHub لإنشاء نسخة خاصة بك.

2. **انسخ المستودع إلى جهازك (Clone)**  
   ```bash
   git clone https://github.com/اسمك/arabic-nlp-toolkit.git
