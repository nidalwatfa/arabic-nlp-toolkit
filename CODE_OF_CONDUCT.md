
# قواعد السلوك للمساهمين في Arabic NLP Toolkit

## هدفنا
نحن ملتزمون بتوفير بيئة مفتوحة ومرحبة وآمنة لجميع المساهمين، بغض النظر عن العمر، الجنس، الخلفية الثقافية، اللغة، أو الخبرة التقنية.

## السلوك المتوقع
- إظهار الاحترام واللطف تجاه الآخرين.
- تقبل وجهات النظر المختلفة بروح إيجابية.
- تقديم النقد البنّاء بطريقة ودية.
- التعاون بروح الفريق.
- الالتزام بمعايير التواصل المهني.

## السلوك غير المقبول
- استخدام لغة مسيئة أو مهينة.
- أي شكل من أشكال التمييز أو التحرش.
- نشر محتوى غير لائق أو غير قانوني.
- تعطيل عمل الآخرين أو المشروع عمدًا.

## المسؤولية
- مشرفو المشروع مسؤولون عن تطبيق هذه القواعد.
- يمكنهم اتخاذ إجراءات مناسبة مثل التحذير أو حظر المساهمين الذين ينتهكون القواعد.

## الإبلاغ
إذا واجهت أي سلوك غير مقبول، يرجى فتح Issue أو التواصل مع مشرفي المشروع بشكل مباشر.
