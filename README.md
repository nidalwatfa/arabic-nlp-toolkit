
# Arabic NLP Toolkit

**Arabic NLP Toolkit** هي مكتبة مفتوحة المصدر تهدف إلى تسهيل مهام معالجة اللغة الطبيعية (NLP) للغة العربية.  
توفر أدوات جاهزة للتطبيع، إزالة كلمات الوقف، التجزئة، التجذير، والتحليل الأساسي للنصوص العربية.

---

## ✨ المزايا
- تطبيع النصوص العربية (Normalization)
- إزالة كلمات الوقف (Stopwords Removal)
- تجزئة النصوص (Tokenization)
- التجذير (Stemming)
- تحليل نصوص أساسي (Basic Text Analysis)

---
