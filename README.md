
# Arabic NLP Toolkit

أداة مفتوحة المصدر لمعالجة النصوص العربية بشكل مبسط.  
تتضمن وظائف أساسية مثل:
- التطبيع (Normalize)
- إزالة كلمات الوقف (Stopwords Removal)
- التجزئة (Tokenization)
- التجذير (Stemming)
- التحليل الأساسي للنصوص (Text Analysis)
